MyPortfolio

MyPortfolio est une application web développée en Python avec le framework Flask, structurée selon une architecture modulaire. Elle sépare distinctement la gestion des routes, les templates HTML, les fichiers statiques (CSS, JavaScript, images) et les données. Pour sécuriser le site, mon choix s'est porté sur les extensions Flask-Talisman (HTTPS, en-têtes de sécurité) et CSRF pour protéger les formulaires contre les attaques intersites... Ce site, sur lequel vous naviguez actuellement, constitue le résultat de ce projet personnel, conçu pour mettre en valeur mes compétences, mes projets, ainsi que mon parcours académique et professionnel de manière claire, interactive et accessible.

Visualisation Rennes Mobilités

Ce petit projet propose une visualisation interactive des mobilités à Rennes en utilisant la bibliothèque Bokeh en Python.
